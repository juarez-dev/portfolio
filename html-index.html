DEFINITIONS

HTML Tag: A String that defines a structure or content block in HTML.

HTML Attribute: Additional characteristics assigned to an HTML tag to define its behavior or appearance.

CSS Stylesheet: Defines the design and layout of a webpage using CSS

HTML TAGS

  • <html>: Starts an HTML document.
  • <head>: Contains metadata about the document(title, stylesheets, etc.).
  • <body>: Contains the visible content of the page.
  • <p>: A text paragraph.
  • <a>: Defines of hyperlink.
  • <img>: An Image.
  • <link>: Defines the relationship between a document and an external resource.
  • <title>: Adds a title to your page, it is part of the head .
  • <hr>: Horizontal Rule, defines a thematic break, is used to separate content or define a change.
  • <br>: Defines a line break.
  • <pre>: Defines a preformatted text, it preserved both spaces and line breaks.

HTML FORMATTING ELEMENTSS

  • <b>: Bold text
  • <storong>: Important text
  • <i>: Italic text
  • <em>: Emphasized text
  • <mark>: Marked text
  • <small>:Smaller text
  • <del>:Deleted text
  • <ins>:Inserted text
  • <sub>:Subscript text
  • <sup>:Superscript text

HTML QUOTATION & CITATION ELEMENTS

  • <blockquote>: Defines a section that is quoted from another source.
  • <q>: Defines a short quotation, normally insert marks arround the quotation
  • <abbr>: Defines an abbreviation or an acronym, it can give useful information for the abb/acro when you mouse over the element.
  • <address>: Defines the contact information for the outhor/owner of a document or an article
  • <cite>: Defines the title of a creative work, the element usually renders in Italic
  • <bdo>: Tag used to override the current text direction.

ADD COMMENTS

  • <!-- comments -->: Can be used to hide content or do a comment.

HTML ATTRIBUTES

  • href: Defines the hyperlink URL (<a>).
  • src: The image path for <img>.
  • alt: Alternative text for an image that cannot be dispayed.
  • class: Used for assing a class to an element for a CSS styling.
  • id: Asings an id to an element.
  • title: Definition of the abbreviation, shows when put on the mouse.<abbr>
  • cite: Define a link for a <blockquote>.
  • dir: rtl -> write right to left <bdo>.

CSS STYLE

            CSS can be added to HTML documents in 3 ways:
                -> Inline: by using the style sttribute inside HTML elements.
                -> Internal: by using a <style> element the <head> section.
                -> External: by using a <link> element to link to an external CSS file.

LINKS TO EXTERNAL CSS

<link rel="stylesheet" href="css link">

  • style="property":Defines a property for an HTML element.
    • color:color: Defines text color.
    • font-family:font: Defines text font.
    • font-size:size: Defines text size.
    • text-align:alignment: Defines text alignment.
    • background:color: Defines background color.
    • border:px solid color: Defines the borders.
    • padding:px: Defines a padding(space) between the text and the border.
    • margin:px: Defines a margin outside the border.

TABLES

Tables allow web developers to arrange data into rows and columns

  • <table>: Defines a table.
  • <th>: Defines a header cell in a table.
  • <tr>: Define a row in a table.
  • <td>: Define a cell in a table.
  • <caption>: Defines a table caption.(subtitle).
  • <colgroup>: Specifies a group of one or more columns in a table for formatting.
  • <col>: Specifies column properties for each column within a colgroup element.
  • <thead>: Groups the header content in a table.
  • <tbody>: Groups the body content in a table.
  • <tfoot>: Groups the footer content in a table.
