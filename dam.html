Higher Technician in Multiplatform Application Development (DAM)

Institution: CIDEAD (Center for Innovation and Development of Distance Education)

Graduation Year: 2025

Level: Higher Vocational Training (EQF Level 5)

Acquired Competencies

  • Develop multiplatform applications with database access.
  • Create efficient and user-friendly graphical user interfaces (GUIs).
  • Use version control systems such as Git.
  • Apply OOP, design patterns, and best practices.
  • Consume web services and REST APIs.
  • Develop mobile apps for Android.
  • Manage SQL and NoSQL databases.
  • Implement security measures in software development.
